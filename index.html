Закрывающий тег

Начало пути: HTML и первый тег

HTML

Помню первый день курса. Открыл редактор, написал <h1> и увидел заголовок в браузере — это было настоящее волшебство. Казалось бы, всего один тег, а сколько эмоций.

Именно тогда я понял: веб — это живое, и каждый элемент имеет смысл.

CSS: когда дизайн оживает

CSS

Первый раз применил цвет через CSS — и страница заиграла красками. Flex, Grid, анимации — за каждым свойством скрывается целый мир.

CSS кажется простым, пока не начинаешь центрировать div.

JavaScript: логика и жизнь

JS

JavaScript открыл дверь в интерактивность. Переменные, функции, события — всё встало на своё место после первого рабочего счётчика.

Когда кнопка реально что-то делает — это уже не вёрстка, это приложение.

Git: история без потерь

GIT

Git — это машина времени для кода. Коммит, ветка, мёрж — теперь работа в команде не страшна, и каждое изменение задокументировано.

Главное правило: делай коммиты чаще, чем думаешь, что нужно.

Адаптивность: один сайт для всех

MEDIA

Mobile first — не просто слова. Когда сайт красиво выглядит на телефоне, планшете и десктопе одновременно, это настоящая победа.

Медиазапросы и clamp() стали лучшими друзьями.

Доступность: вёрстка для всех людей

A11Y

Семантика, alt-тексты, aria-метки — это не лишние детали, а забота о пользователях. Хорошая вёрстка должна работать для всех.

Скринридер — лучший тест на правильную структуру страницы.

SVG и анимации: движение в вёрстке

SVG

SVG-графика, CSS-анимации, keyframes — страница перестала быть статичной. Теперь элементы живут, двигаются и реагируют на пользователя.

Иконка-сердце с анимацией лайка — маленький шедевр этого проекта.

Финал: закрывающий тег пути

THE END

Этот проект — итог большого пути. Каждый освоенный инструмент нашёл своё место: семантика, стили, скрипты, шрифты, анимации.

Закрывающий тег — не конец, а начало. Теперь впереди настоящие проекты и реальная работа. </учёба>
