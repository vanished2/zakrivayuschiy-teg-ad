# Закрывающий тег

Финальный проект модуля вёрстки курса Яндекс Практикума.

## О проекте

Сайт-галерея карточек воспоминаний, посвящённых пути обучения веб-разработке. На странице отображаются карточки с личными воспоминаниями о каждом этапе курса, стилизованные CSS-фильтрами и анимациями.

## Функциональность

- Адаптивная вёрстка (десктоп и мобильные устройства от 375px)
- Карточки с уникальными CSS-фильтрами
- Анимированная иконка сердца с состояниями hover, active и liked
- Кнопки с эффектом mix-blend-mode и псевдоэлементом на hover
- Модальное окно на основе элемента dialog
- Фиксированный фоновый паттерн из градиентов
- Вариативный шрифт Inter и акцентный шрифт PressStart2P

## Технологии

- HTML5 (семантическая разметка)
- CSS3 (переменные, flex, grid, анимации, трансформации, mix-blend-mode, backdrop-filter)
- Вариативные шрифты
- SVG (инлайн и symbol + use)
- Vanilla JavaScript
